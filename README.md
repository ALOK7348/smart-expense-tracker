# Smart Expense Tracker

A full-stack Smart Expense Tracker application built using HTML, CSS, JavaScript, Node.js, Express.js, and MySQL.

## Features

- Add expenses
- View expenses dynamically
- Update expenses
- Delete expenses
- Category filtering
- Total expense dashboard
- Responsive and attractive UI

## Tech Stack

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Node.js
- Express.js

Database:
- MySQL

## Project Architecture

Frontend
↓
Fetch API
↓
Express Routes
↓
Controllers
↓
MySQL Database

## API Endpoints

POST /api/expenses/add

GET /api/expenses/all

PUT /api/expenses/update/:id

DELETE /api/expenses/delete/:id
